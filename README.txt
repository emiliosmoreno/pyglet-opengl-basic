Requisitos previos:
1.- Verificar la instalación de python:

Abrir la consola cmd en windows y ejecutar el siguiente comando: python --version. 

Si se ejecuta correctamente, aparecerá algo parecido a esto:

Python 3.11.7

2.- Instalar el módulo PyGame

Desde la consola, lanzar el siguiente comando: pip install PyGame

3.- Instalar el módulo PyOpenGL

Desde la consola, lanzar el siguiente comando: pip install PyOpenGL

4.- Instalar el módulo PyOpenGL_accelerate

Desde la consola, lanzar el siguiente comando: pip install PyOpenGL_accelerate

5.- Ejecutar el fichero 01_entorno_basico.py, y comprobar que se ejcuta correctamente.
